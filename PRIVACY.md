# 隐私说明

生效日期：2026-07-22

HarmonyWireguard 不提供账号、广告、分析或远程配置服务，也不会把 WireGuard 配置上传到项目维护者的
服务器。

## 本地处理的数据

为建立用户主动配置的 VPN 隧道，应用会在设备上处理：

- 隧道名称、接口地址、DNS、MTU 和路由范围；
- 服务端 Endpoint、公钥、私钥和可选的预共享密钥；
- 连接状态、握手时间、收发字节数和本地诊断日志。

当前技术预览版将配置保存在应用私有沙箱内。数据不会主动离开设备，但尚未迁移到硬件支持的密钥存储；
请仅在受信任的测试设备上使用。

## VPN 流量与 DNS

启用隧道后，系统会按照用户导入的配置把匹配流量发送到指定 WireGuard 服务端，并使用配置中的 DNS。
服务端、DNS 提供方和目标网站可能按各自政策处理网络元数据。项目维护者不运营中转服务器，也无法访问
用户的 VPN 流量。

## 文件、二维码与剪贴板

- 文件导入和二维码扫描只在用户主动选择时启动系统选择器或 Scan Kit。
- 应用不会在后台读取剪贴板。用户主动复制密钥或诊断信息时，内容会写入系统剪贴板；敏感字段支持定时
  清理，但其他应用或系统组件在清理前仍可能读取它。

## 保存与删除

配置会保留到用户在应用中删除，或清除应用数据/卸载应用。诊断记录用于当前设备上的界面展示，不会上传。

## 权限用途

- 网络访问与网络状态：解析 Endpoint、建立 WireGuard UDP 连接并感知网络变化。
- 隐私窗口：在用户启用相关设置时阻止应用内容被截屏或录屏。

## 联系方式

隐私问题可通过 GitHub 仓库 `JavaZeroo/HarmonyWireguard` 联系维护者。报告问题时请勿附带真实私钥、
预共享密钥或完整配置文件。
