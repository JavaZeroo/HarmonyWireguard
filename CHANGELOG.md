# Changelog

本项目遵循语义化版本命名。

## [1.0.0] - 2026-07-22

首个公开技术预览版本。

### Added

- HarmonyOS NEXT 原生 WireGuard VPN Extension 与 C/C++ 数据通道。
- WireGuard 配置导入、编辑、列表和详情体验。
- 握手状态、流量统计、连接诊断与运行日志。
- 手机、横屏、平板和 2-in-1 自适应布局。
- 深浅色、紧凑模式与隐私窗口设置。

### Changed

- 重构隧道页导航和宽屏主从布局。
- 减少流量图、日志面板和诊断状态的无效刷新。
- 移除未引用页面、组件、旧服务和生成目录。
- 更正第三方组件许可展示，补齐发布文档与隐私说明。

### Fixed

- 正确保留 `PersistentKeepalive = 0`，仅为旧配置的缺失值提供默认值。
- 严格校验 VPN 状态事件，并限制事件发布与订阅到应用 Bundle。
- 启动连接后立即发布运行状态，避免首个定时周期前界面滞后。
- 修复横屏尺寸换算、握手过期状态和详情日志刷新问题。
- 移除逐包日志与明文端点日志。

### Known limitations

- 本版本为技术预览，尚未完成独立密码学与安全审计。
- IPv6 Endpoint、Address 与 DNS 尚未完整支持。
- 配置当前保存在应用私有沙箱中，尚未迁移到硬件支持的密钥存储。
